### **Screen Recording Setup**
```
Software: OBS Studio (free) or Camtasia
Resolution: 1920x1080
Frame Rate: 30 fps
Bitrate: 5000 kbps
Format: MP4 (H.264)
```

### **Voiceover Recording**
```
Microphone: USB condenser mic (Blue Yeti, Audio-Technica AT2020)
Software: Audacity (free) or Adobe Audition
Format: WAV 48kHz 16-bit
Remove background noise
Normalize audio to -3dB
```

### **Music**
```
Source: Epidemic Sound, Artlist, or YouTube Audio Library
Style: Upbeat tech/corporate
Tempo: 120-140 BPM
Volume: -20dB (background, don't overpower voiceover)
License: Royalty-free
```

### **Editing**
```
Software: DaVinci Resolve (free), Adobe Premiere, or Final Cut Pro
Transitions: Quick cuts, smooth fades
Text: Modern sans-serif (Inter, Roboto)
Color grading: Slightly vibrant, tech-feel
Export: MP4, H.264, 1080p, 30fps
```
